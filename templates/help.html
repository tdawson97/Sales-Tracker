FAQ

  • How can I add an employee to my team?
    • On the Employees tab, click on the Add button and enter in all the required information.
  • How can I add an item to my inventory?
    • On the Items tab, click on the Add button and enter in all the required information.
  • How can I edit an employee or item entry?
    • On the appropriate tab, search for the desired entry in the table or by using the search field. Select the employee or item to be changed to view the profile and select the Edit button.
  • How can I delete an employee or item entry?
    • On the appropriate tab, search for the desired entry in the table or by using the search field. Select the employee or item to be changed to view the profile and select the Delete button.
  • How can I filter the sales report?
    • On the sales tab, you can search for an employee name, item name, or order # to generate a new sales report with only your serach item.
